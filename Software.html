Pseudo Code

If (no new key reading or touching with the RFID Scanner)return;    //reset state the shortest way to close the door especially when the door opens accidentally

Scan the key from the user;
Display the key identification number to the LCD; 
If (the key is reading correctly){
If (worker's RFID reading is high && customer’s RFID reading is low){
LCD displays = worker’s LCD displays wait for 0.05 seconds; The blue light & the laser light is on and the stepper motor door lock from the worker side is opened;
If the laser light is being covered( by the food) and the photoresistor is reading low{
Wait for 10ns;
}
The worker side’s door is closing and the customer side’s LCD is saying the food is ready;
Reset Worker’s LCD;
}

else if(customer’s RFID reading is HIGH && worker’s RFID reading is LOW){
If (the FSR customer’s side or FSR worker’s side is equal to low){
Customer side LED on;
RED and GREEN LED on;
wait for 1000ns;
display empty on the customer’s LCD;
wait for 3000ns;
reset the LCD;
}

else {
Customer side LED on;
RED and GREEN LED on;
wait for 1000ns;
The customer’s side door is open;
while(the both FSR reading is HIGH){
Wait for 1000ns; (to let the customer take out the food)
}
the customer’s side door closing;
reset the LED & LCD;
}
}
Else if(the customer’s RFID && the worker’s RFID reading is equal to HIGH || the FSR for customer && FSR for the worker is reading LOW){
ask the user to try scanning the key again;
wait for 3000ns;
reset LCD;
}
}
else if (the key is reading not correctly){
Display incorrect order for both sides;
wait for 3000ns;
reset LCD;
}
