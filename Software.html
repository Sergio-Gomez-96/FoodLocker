State Table

State Event Action
S1: Locker waits for input Worker scans RFID card Worker side door opens
S2: Locker is loaded with customers order FSR sensor is HIGH Door closes after 10 seconds
S3: Locker waits for input Customer scans RFID card Customer side door opens
S4: Customer takes their order FSR sensor is low Customer side door is closed
S5: Locker is empty Time since door closed is 5 seconds UV lights activate

Pseudo code:

             Set up RFID pins

             Set up LCD pins

             Set up RGB LED pins

             Set up Photoresistor, Force Sensing Resistor pins


             Food lock close from both side

             UV lights on if FSR sensor is LOW

             Wait for Photoresistor to detect for 5 sec

             Door open (Workers)

             Check for inputs from RFID(Worker side RFID is HIGH)

             Door lock close for 10sec(Workers) if FSR sensor is HIGH

             Wait for customer to Enter Code for RFID(Customer side RFID is HIGH)

             Door Lock open(Customer)

             Back to beginning position
